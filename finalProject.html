So my partner and fellow CSC 106 student Domenic programmed the game you see below titled "In The Bin".

The game gives the user access to a moveable chest controlled using the left and right arrow keys on their computer.

The user has to guide chest across the screen to collect different colored gems falling from the sky to earn points.

To provide a bit of challenge, the user must avoiod colliding with bombs falling from the sky which will cost them points.
